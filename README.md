# E-commerce Sales Power BI Dashboard

This project is a Power BI template designed to analyze e-commerce sales performance and business metrics.

## Project Overview
The dashboard provides insights into sales trends, revenue performance, product categories, and customer behavior across different time periods.

## Files
- Power BI Template: /dashboard/E.commercesales.pbit
- Dataset: /dataset/
- Report / Screenshots: /report/

## Tools Used
- Power BI
- Excel
- Data Analysis

## Key Insights
- Sales and revenue trends
- Category and product performance
- Customer and regional analysis
